🤖 튜브워커 (TubeWorker)
오토워커 자동 영상 제작 프로그램. 대본 → 이미지 → 음성 → 편집을 자동화. 크레딧 차감 방식
🎨 위스크 (Whisk)
Google AI 이미지 생성 도구. 튜브워커에서 이미지 생성에 사용 중. 서비스 종료 예정 → 플로우로 전환
🌊 플로우 (Flow)
Google 차세대 이미지 생성 도구. 위스크의 후속. 튜브워커에 통합 예정
🎬 그록 (Grok)
xAI의 AI 영상 생성 도구. 가성비 최고지만 자동화를 막아놔서 수동 사용 필요. 유료 전환됨
🎙️ 일레븐랩스 (ElevenLabs)
AI 음성 합성(TTS) 서비스. 개인 구독보다 강사 제공 API가 저렴. V2(일관성) / V3(발음) 모델 선택 가능
✂️ 캡컷 (CapCut)
영상 편집 프로그램. 튜브워커에서 생성한 프로젝트를 CapCut Drafts 폴더에 넣어 편집
🧠 클로드 코드 (Claude Code)
Anthropic의 AI 코딩/작업 도구. 스킬과 에이전트를 사용해 대본 작성, 파일 관리 등을 자동화
⚡ 스킬 (Skill)
클로드에서 반복 작업을 자동화하는 사전 정의 명령. SKILL.md 파일에 워크플로 정의. 게임 스킬처럼 장착해서 사용
👤 에이전트 (Agent)
특정 역할을 수행하는 AI 서브워커. 예: video-analyst(영상 분석), data-researcher(팩트체크), strategist(전략)
🪙 크레딧
튜브워커 내 사용량 단위. 대본 길이/이미지 수에 따라 차감. 구글 폼으로 충전 신청 (결제 시스템 준비 중)
🎥 롱폼
5분 이상의 긴 유튜브 영상. 수익창출의 주 포맷. 하루 2~4개 업로드 권장
📱 숏폼 / 숏츠
60초 이내의 짧은 영상. 롱폼 보조 수단으로 병행 권장. 수익화 집중 시 하루 10개까지도
💰 수창 / 수익창출
유튜브 파트너 프로그램 수익화 조건 달성. 1차: 구독 500 + 시청 3000시간, 2차: 구독 1000 + 시청 4000시간
🎣 후킹 인트로
시청자를 잡아두는 영상 도입부 (처음 3~5초). 체류시간에 직접적 영향 → 알고리즘 노출 결정
🎞️ 스토리보드
영상의 장면별 구성 계획. 튜브워커에서 자동 생성되나, AI가 문맥에 맞는 씬을 못 찾는 경우도 있음
🧱 벽타기
카카오톡 채팅방에서 지나간 대화를 위로 스크롤하며 읽는 것. 사담방 특성상 메시지가 많아 벽타기가 힘듦
👑 프프 / 프리미엄반
추가 심화 강의반. 별도 참여 코드 필요. 오토워커 1기 기본반과 별개 운영
